🌊 鳴潮矩陣疊兵 (矩陣) 編隊工具
這是一款專為《鳴潮》(Wuthering Waves)「矩陣疊兵 (矩陣)」模式設計的網頁版配隊輔助工具。
本工具當前採用友人提供0+1 DPS 期望值數據庫，能幫助玩家根據自身的角色池與操作熟練度，輕鬆規劃出最佳的 16 關極限通關陣容！

✨ 核心功能特色 (Initial Public Release - v4.5.1)
🎯 動態 DPS 推薦引擎
打破傳統靜態表格的限制！無論你先選定主輸出、副C還是生存位，系統都會自動計算並將「期望值最高」的最佳隊友置頂推薦。

⚔️ 客製化排軸與難度系統 (獨家功能)
內建多排軸數據庫，包含明確的難度標籤（🟩輪椅 / 🔵中等 / ⭐進階 / ⚠️極難）。玩家可自由在「配置實戰排軸」面板中取消勾選自己不會打的極限手法，系統將只採用你真實能打出的 DPS 進行智能排序，告別虛假的理論上限！

🎒 角色與篩選
只需勾選你擁有的角色，系統就會自動隱藏所有你無法組建的陣容與排軸，還給你一個極度乾淨、無雜訊的編隊舒適。

💾 無縫存檔體驗
內建 LocalStorage 自動記憶功能。網頁關閉、不小心重新整理，你辛苦排好的隊伍與勾選進度都會完整保留。

📷 一鍵匯出分享圖
排班完成後，點擊按鈕即可自動過濾掉空白關卡，一鍵生成精美、乾淨的「16關實戰排班圖」，方便保存或分享至各大社群論壇。

⚡ 運作方式
當前採用純前端 Vanilla JS 與高效的 DOM 渲染機制打造，無須安裝、無須伺服器，當前版本可直接利用瀏覽器即可使用。
