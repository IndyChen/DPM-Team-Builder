# 鳴潮矩陣編隊工具 - 完整版本更新歷史 (Changelog)

本專案的所有顯著更改、功能新增與錯誤修復，都將記錄於此文件中。

### 🏷️ 關於版本命名規範 (Versioning Convention)
本專案自 `v4.8.1` (後重構) 起，正式採用 **[語意化版本控制 (Semantic Versioning)](https://semver.org/lang/zh-TW/)** 規範，格式為 `主版號.次版號.修訂號 (Major.Minor.Patch)`：
* **Major (主版號)**：進行不相容的 API 修改、底層架構大翻新。
* **Minor (次版號)**：以向下相容的方式新增功能。
* **Patch (修訂號)**：以向下相容的方式修復 Bug，或純粹進行數值更新。

---
##[v4.6.2] - 2026-03-14 
###全面優化與校正 (Optimization & Fixes)
- 🐞 重整 UI 修復：修復網頁重新整理後，「清空/全選排軸」按鈕未能正確讀取 LocalStorage 狀態，導致全空時按鈕仍顯示紅色的 Bug。
- 🔍 排軸快搜實裝：在「配置實戰排軸」區塊新增專屬搜尋框，支援簡繁與部分匹配，找特定角色的排軸更有效率。
- 📊 數據全面校正：依照最新數據，更新卡卡羅、菲比、愛彌斯等所有隊伍的排軸與 DPS 數值。
- ⚔️ 漂泊者型態分離：配隊選單正式將漂泊者分離為「光主、暗主、風主」。三者共用同一個「漂泊者」的耐力上限，因遊戲當前如此設定！

## [v4.6.1] - 2026-03-13
###體驗升級 (QoL Improvements)
- 🔍 簡繁無縫部分搜索：新增角色搜尋框，支援部分匹配（如輸入「千」即可），並內建字典，允許輸入簡體字（如「卡卡罗」）也能準確找到繁體角色。
- ⚡ 常駐排軸快篩：於排軸面板新增常駐的 5 顆快捷鍵。總開關「一鍵全選/清空」確保放置於首位，搭配 4 顆特定難度（🟩/🔵/⭐/⚠️）獨立開關，極大化過濾效率。

## [v4.5.1] - 2026-03-13
### 🐞 緊急修復 (Hotfixes)
- **核心崩潰修復**：修復了因底層陣列變數命名衝突，導致網頁載入瞬間觸發 `ReferenceError` 的白畫面當機 Bug。確保「配置實戰排軸」面板與「剩餘可用次數」追蹤器能 100% 穩定渲染。

## [v4.5.0] - 2026-03-13
### 🚀 重大更新 (Major Features)
- **自訂排軸資料庫面板**：新增獨立的「配置實戰排軸」區塊。系統會自動隱藏玩家未擁有的隊伍，並允許玩家精確取消勾選自己不會打的排軸（例如極難的劍切軸）。下拉選單將動態連動，僅根據玩家勾選的排軸提供最真實的 DPS 期望值。
- **一號位全解禁**：移除了以往一號位（主輸出）只能看見系統推薦名單的限制，現在玩家可於「其他角色」清單中自由選擇所有已擁有的角色作為主 C。
- **實用工具回歸**：在保證底層效能的前提下，重新加回 `LocalStorage` 自動記憶存檔功能，以及基於 `html2canvas` 的一鍵匯出排班圖片功能。

## [v4.4.0] - 2026-03-12
### 🛡️ 效能重構與介面優化 (Performance Refactor & UI)
- **重返穩健骨幹**：因前述實驗性版本（動態全過濾）造成嚴重卡頓與無限渲染衝突，全面回退至最穩定的 `v4.3.1` 原生架構進行升級，徹底解決下拉選單效能災難。
- **介面潔淨化**：將第一區塊「勾選擁有角色」改為 `<details>` 預設折疊面板，大幅提升初始載入時的視覺整潔度，並完美保留原生的「一鍵全選 / 清空」按鈕手感。
- **難度視覺化**：為 DPS 數據庫導入直觀的難度標籤（🟩輪椅 / 🔵中等 / ⭐進階 / ⚠️極難）。

## [實驗版除錯與擴充期] - 2026-03-12
*(註：此階段包含 v2.0.0 ~ v2.1.2 短暫的版號重構)*
- **快捷按鈕**：於角色勾選區塊新增「✅ 全選 / ❌ 全取消」快捷鍵。
- **動態背包過濾**：嘗試導入極度嚴格的動態過濾機制，未勾選擁有的角色不再顯示其對應排軸（後因效能瓶頸於 v4.4.0 放棄此架構，改為現在的排軸面板）。
- **追蹤器修復**：解決動態過濾機制中舊變數引發的面板空白問題。

## [v4.3.1] - 2026-03-10
### ⚙️ 核心引擎穩定版 (Core Engine)
- **DOM 快取優化**：重寫下拉選單的生成與更新邏輯，導入防閃爍與快取機制，帶來極致流暢、不卡頓的操作體驗。

## [v4.3.0] - 2026-03-10
### 🧠 關聯推薦升級 (Smart Recommendations)
- **全向關聯引擎**：打破傳統選角順序限制！無論先選定副 C 或是生存位，系統皆能反向推算並推薦出最佳匹配的主輸出。
- **定位指引置頂**：在隊伍空位狀態下，選單會自動將有能力擔任該定位（主C/副C/生存）的最佳角色置頂顯示。

## [早期版本 v1.0.0 - v4.1.0] - 2026年3月上旬
### 🏗️ 基礎建設與成型 (Foundations & UI/UX)
- **推薦系統上線 (v4.1.0)**：整合近百組 DPS 期望值陣容，新增「一鍵套用」下拉選單功能。
- **視覺大翻新 (v4.0.0)**：全面導入深色電競 UI 介面（Dark Mode），實裝 Sticky 浮動追蹤面板與快捷清空按鈕。
- **次數追蹤系統 (v3.0.0)**：首次導入《鳴潮》矩陣疊兵模式的疲勞值機制，並實裝「紅色警告邊框」防呆提示。
- **動態背包連動 (v2.0.0)**：建立角色池勾選區，讓下拉選單僅顯示玩家實際擁有的角色。
- **靜態骨架建立 (v1.0.0)**：建立 16 關表格佈局與基礎三定位框架。
